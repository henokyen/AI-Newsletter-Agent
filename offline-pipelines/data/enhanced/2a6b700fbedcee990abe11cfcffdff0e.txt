# datasketch: Big Data Looks Small
[](https://github.com/ekzhu/datasketch/#datasketch-big-data-looks-small)
[![https://static.pepy.tech/badge/datasketch/month](https://camo.githubusercontent.com/43d67355304af7750aed66548442cc8042ec7e5bee87ead5f4d5bb1923b61a64/68747470733a2f2f7374617469632e706570792e746563682f62616467652f64617461736b657463682f6d6f6e7468)](https://pepy.tech/project/datasketch) [![](https://camo.githubusercontent.com/bbe32b803fd2be86507a5538346ebabff55fab129f6636113e4271e666f7ea4d/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e3539383233382e737667) ](https://zenodo.org/doi/10.5281/zenodo.598238)
datasketch gives you probabilistic data structures that can process and search very large amount of data super fast, with little loss of accuracy.
This package contains the following data sketches:
Data Sketch | Usage  
---|---  
[MinHash](https://ekzhu.github.io/datasketch/minhash.html) | estimate Jaccard similarity and cardinality  
[Weighted MinHash](https://ekzhu.github.io/datasketch/weightedminhash.html) | estimate weighted Jaccard similarity  
[HyperLogLog](https://ekzhu.github.io/datasketch/hyperloglog.html) | estimate cardinality  
[HyperLogLog++](https://ekzhu.github.io/datasketch/hyperloglog.html#hyperloglog-plusplus) | estimate cardinality  
The following indexes for data sketches are provided to support sub-linear query time:
Index | For Data Sketch | Supported Query Type  
---|---|---  
[MinHash LSH](https://ekzhu.github.io/datasketch/lsh.html) | MinHash, Weighted MinHash | Jaccard Threshold  
[MinHash LSH Forest](https://ekzhu.github.io/datasketch/lshforest.html) | MinHash, Weighted MinHash | Jaccard Top-K  
[MinHash LSH Ensemble](https://ekzhu.github.io/datasketch/lshensemble.html) | MinHash | Containment Threshold  
[HNSW](https://ekzhu.github.io/datasketch/documentation.html#hnsw) | Any | Custom Metric Top-K  
datasketch must be used with Python 3.7 or above, NumPy 1.11 or above, and Scipy.
Note that [MinHash LSH](https://ekzhu.github.io/datasketch/lsh.html) and [MinHash LSH Ensemble](https://ekzhu.github.io/datasketch/lshensemble.html) also support Redis and Cassandra storage layer (see [MinHash LSH at Scale](http://ekzhu.github.io/datasketch/lsh.html#minhash-lsh-at-scale)).
## Install
[](https://github.com/ekzhu/datasketch/#install)
To install datasketch using `pip`:
```
pip install datasketch

```

This will also install NumPy as dependency.
To install with Redis dependency:
```
pip install datasketch[redis]

```

To install with Cassandra dependency:
```
pip install datasketch[cassandra]

```
