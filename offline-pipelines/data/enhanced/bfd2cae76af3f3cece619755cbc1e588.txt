## Getting started
[](https://github.com/FrancescoSaverioZuppichini/python-template/#getting-started)
### Installation
[](https://github.com/FrancescoSaverioZuppichini/python-template/#installation)
To set it up and run
```
uv sync
```

Then
```
python main.py
```

Will output a random joke
```
Why did the cow in the pasture get promoted at work? ... Because he is OUT-STANDING in his field!

```

You can now run, for example, a function defined as `scripts` in the [`pyproject.toml`](https://github.com/FrancescoSaverioZuppichini/python-template/blob/main/pyproject.toml)
```
make_me_laugh
```

### Linting
[](https://github.com/FrancescoSaverioZuppichini/python-template/#linting)
```
ruff check

```

### Formatting
[](https://github.com/FrancescoSaverioZuppichini/python-template/#formatting)
```
ruff format

```

## CI/CD
[](https://github.com/FrancescoSaverioZuppichini/python-template/#cicd)
### Tests
[](https://github.com/FrancescoSaverioZuppichini/python-template/#tests)
Tests inside `/tests` are run using [`pytest`](https://docs.pytest.org/en/stable/) on PR both on `dev` and `main`
### Publish Package
[](https://github.com/FrancescoSaverioZuppichini/python-template/#publish-package)
In order to publish to [pypi](https://pypi.org/) you need to create a publisher on pypi.
This is explained [here](https://packaging.python.org/en/latest/guides/publishing-package-distribution-releases-using-github-actions-ci-cd-workflows/) and [here](https://docs.pypi.org/trusted-publishers/)
In practice go your pypi project -> Manage Project -> Publishing, scroll and "add a new publisher"
### Docker
[](https://github.com/FrancescoSaverioZuppichini/python-template/#docker)
[`Dockerfile`](https://github.com/FrancescoSaverioZuppichini/python-template/blob/main/Dockerfile) contains a multi stage build that uses `--compile-bytecode` to compite your package. For this example, the resulting image is just
```
docker build -t python-template .
```

```
REPOSITORY    TAG    IMAGE ID    CREATED     SIZE
python-template  latest  1ded7d260b1c  58 seconds ago  55.4MB

```

The image is build using the [`build`](https://github.com/FrancescoSaverioZuppichini/python-template/blob/main/.github/workflows/build.yml) workflow when a new _release_ is created
